Pipecut provides a data transformation library with a highly interactive user interface.

The back-end library is in the process of being disentangled from the front-end code.

Proof of concept code was demonstrated Nov 1, 2014 at MeetBSD California.

0.5.0-alpha was demonstrated Feb 21st, 2015 at SCALE 13x
